*사용자 정보
Git을 설치하고 나서 가장 먼저 해야 하는 것은 
사용자이름과 이메일 주소를 설정하는 것이다.
 Git은 커밋할 때마다 이 정보를 사용한다. 한 번 커밋한 후에는 정보를 변경할 수 없다.
$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com(구글링)

확인방법
$ git config --global --list
